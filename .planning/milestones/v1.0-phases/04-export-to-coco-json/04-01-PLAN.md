<threat_model>
## STRIDE Notes

- **Information disclosure:** Export endpoint is authenticated and includes only dataset filenames plus annotation metadata, not raw DICOM metadata.
- **Tampering:** Export is read-only and generated from ORM state.
- **Repudiation:** Export includes annotator identity and timestamps for traceability.
- **Denial of service:** Request-time generation is acceptable for v1; implementation should stream/serialize efficiently enough for ~1600 images.
</threat_model>
